
## font-awesome

Add a gem to your `Gemfile`
```
gem 'font-awesome-rails'
```

then install it using `bundle install`.

## usage
### css
include this to your `application.css`
```css
/*
 *= require font-awesome
 */
```

### sass
include this to your `application.scss`
```scss
/*
 *= require font-awesome
 */
```
